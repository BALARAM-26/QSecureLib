# User Guide

## Generate Keys

1. Open QSecure
2. Click Generate Keys
3. Enter password

Generated files:

- alice.pub
- alice.priv.enc
- sign.pub
- sign.priv.enc

## Encrypt File

1. Browse File
2. Click Encrypt

Output:

filename.qsec

## Decrypt File

1. Select .qsec file
2. Click Decrypt
3. Enter password

Output:

filename.qsec.dec

## Sign File

1. Select file
2. Click Sign
3. Enter password

Output:

filename.sig

## Verify File

1. Select file
2. Click Verify

Result:

Signature VALID
or
Signature INVALID
